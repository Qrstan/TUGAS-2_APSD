Notasi Algoritma Deskriptif

> Membuat sebuah program menghitung nilai rata-rata siswa.

Diketahui : Data siswa berupa; NIM, Nilai Tugas I, Nilai Tugas II, dan Nilai Tugas III.
Ditanyakan : Nilai rata-rata seorang siswa?
Langkah Penyelesaian :
	1. Siapkan data siswa berupa NIM dan Nama Siswa,
	2. Masukkan data NIM dan Nama Siswa pada program tersebut, 
	3. Lengkapi bagian Nilai Tugas dari Tugas I - III; nantinya Nilai Tugas keseluruhan akan dibagi 3 oleh program tersebut,
	4. Cek kembali data seperti NIM, Nama Siswa dan Nilai Tugasnya agar program bisa menampilkan hasilnya sesuai dengan NIM dan Siswa,
	5. Program nantinya akan menampilkan hasil berupa Nilai Rata-rata Siswa dengan NIMnya masing-masing,
	6. Selesai.

Notasi Algoritma Pseudocode

> Membuat sebuah program menghitung nilai rata-rata siswa.

Nama Siswa : string
NIM Siswa : integer
Nilai Tugas I : integer
Nilai Tugas II : integer
Nilai Tugas III : integer
Nilai Rata-rata : integer

read(NIM Siswa, Nama Siswa, Nilai Tugas I, Nilai Tugas II, Nilai Tugas III, Nilai Rata-rata)
Nilai Rata-rata = (Nilai Tugas 1 + Nilai Tugas II + Nilai Tugas III) /3
write(NIM Siswa, Nama Siswa, Nilai Rata-rata)
end
